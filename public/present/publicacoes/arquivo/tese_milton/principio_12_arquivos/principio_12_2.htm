12.2 – Alguns conceitos introdutórios

Em qualquer mecanismo, após a montagem, as posições ou dimensões atuais de alguns dos seus componentes funcionais não são as mesmas posições ou dimensões nominais, conforme projetado. Assim, alguma providência deve ser tomada para as posições (ou formas) de alguns dos componentes sejam alteradas, de modo que o mecanismo possa desempenhar sua função conforme previsto no projeto. Essa ajustagem pode ser feita de diversas formas, seja através de alterações de posições ou remoção de material, ou através do emprego de algum mecanismo auxiliar que execute essas alterações necessárias nas posições dos componentes. Esse mecanismo é chamado de “dispositivo de ajustagem”.

Cada dispositivo de ajustagem possui um ponto inicial de movimento e um elemento final que é representado pelo componente funcional ou pelo elemento a ser movido. Este elemento final é chamado de “unidade de ajustagem”. Na figura 43 é representado um esquema contendo os elementos funcionais de um dispositivo de ajustagem.

 

 

 

Figura 43 - Elementos funcionais de um dispositivo de ajustagem.

 

Geralmente, o ajuste de um sistema, seja este simples ou complicado, consiste em vários processos de ajustagem. Sendo assim, uma série de unidades de ajustagem deverá ser empregada. A unidade de ajustagem possui um conceito amplo, indo desde um dispositivo empregado em um componente específico, até um ajuste a ser feito no sistema como um todo.

Quando se considera o emprego de movimentos relativos, deve-se fixar uma origem como referência para este movimento. Nem sempre esta origem será a mesma para todos os movimentos necessários para ajustagem das submontagens do sistema.
