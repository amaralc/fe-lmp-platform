12 – Ajustagem de mecanismos de precisão

Muitos projetos relacionados a instrumentos de precisão e outras áreas especiais podem ter um resultado insatisfatório se o projetista não possui o conhecimento adequado dos princípios básicos da ajustagem.

As dimensões de peças manufaturadas sempre serão diferentes das dimensões nominais especificadas no projeto detalhado, ficando dentro de uma faixa de tolerâncias. Os erros das dimensões de cada componente de um sistema contribuem para o erro global nesse sistema. O somatório das diferenças de diversos componentes de uma montagem levam a erros globais significativos. Esses erros não podem ser desconsiderados, mas podem ser mantidos dentro de limites toleráveis. Por outro lado, eles podem ser anulados de modo a não interferirem na função do sistema, sendo imperceptíveis, quando utilizadas técnicas de ajustagem.

Como exemplo, consideremos um resultado teórico esperado para a função de um sistema denominado Rte. Se o sistema resulta em um valor Rt, então existe um desvio na função dado pela equação 15:

              DR = Rt  - Rte                                                                                          (15)

Esse desvio deve ser mantido o menor possível, ou pelo menos suficientemente pequeno para o propósito no qual o sistema se destina. Para reduzir DR a um valor aceitável, normalmente são necessárias técnicas de fabricação dos componentes caras e refinadas.

Como a produtividade e o custo de produção limitam o grau de precisão obtenível na fabricação dos componentes, outros meios devem ser encontrados para que se atinja um DR aceitável. A ajustagem oferece esta alternativa, já que permite que imprecisões de fabricação sejam toleradas no cumprimento da função final do sistema.

A aplicação correta das técnicas de ajustagem em uma máquina-ferramenta pode diminuir ou até excluir a necessidade de se aplicar compensação de erros através dos acionamentos da própria máquina, já que a ajustagem vai posicionar os sistemas da máquina de modo mais fiel possível às dimensões relativas indicadas no projeto.
